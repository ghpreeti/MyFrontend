Container

The .container class provides a responsive fixed-width container, meaning its maximum width is determined by the device's screen size.

Container Fluid

The .container-fluid class provides a full-width container, spanning the entire width of the viewport.

Container Small

The .container-sm class provides a responsive fixed-width container, up to a maximum width of 540px.

Container Medium

The .container-md class provides a responsive fixed-width container, up to a maximum width of 720px.

Container Large

The .container-lg class provides a responsive fixed-width container, up to a maximum width of 960px.

Container Extra Large

The .container-xl class provides a responsive fixed-width container, up to a maximum width of 1140px.

Container Extra Extra Large

The .container-xxl class provides a responsive fixed-width container, up to a maximum width of 1320px.
